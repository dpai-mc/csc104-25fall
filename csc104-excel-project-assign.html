Assignment Instructions
  1. Set your color theme to “Office 2013-2022”
  2. Merge cells A1 through F1 and center the title "Coastal Coffee Shop"
  3. Format the title with Font: Arial Black, Size: 18pt, Color: Dark Blue
  4. Merge cells A2 through F2 and center the subtitle
  5. Format the subtitle with Font: Arial, Size: 12pt, Color: Green Accent 6 Darker 50%, Style: Italic
  6. PART 2: Column Headers and Data Formatting (10 points)
  7. Apply the "Heading 3" cell style to the range A4:F4 (column headers)
  8. Center align all column headers
  9. Format all sales data (B5:D9) with Accounting Number Format, 0 decimal places
  10. Bold the product category names in column A (A5:A9)
  11. PART 3: Formulas - Quarter Total (12 points)
  12. In cell E5, create a formula to calculate the Quarter Total for Hot Beverages (sum of October, November, and December)
  13. Copy this formula down through E9 for all product categories
  14. Format column E (E5:E9) with Accounting Number Format, 0 decimal places
  15. In cell E10, create a formula to sum all Quarter Totals (E5:E9)
  16. PART 4: Formulas - Bonus Calculation (15 points)
  17. The owner offers a bonus to product categories that exceed $15,000 in quarterly sales. The bonus is 5% of the quarter total. These values are stored in cells B20 and B21.
  18. In cell F5, create an IF formula that: Awards a bonus (Quarter Total * $B$21) if the Quarter Total >= $B$20, Otherwise displays 0
  19. Use absolute cell references for B20 and B21 so the formula can be copied
  20. Copy the formula from F5 down through F9
  21. Format column F (F5:F9) with Accounting Number Format, 2 decimal places
  22. In cell F10, create a SUM formula to total all bonuses earned
  23. PART 5: Total Row Formatting (8 points)
  24. Apply the "Total" cell style to the range A10:F10
  25. Add a top border (your choice of style) to cells B10:F10
  26. Ensure all values in row 10 display with appropriate number formatting
  27. PART 6: Statistical Analysis (12 points)
  28. Format cell A13 with Font: Arial, Size: 12pt, Bold
  29. In cell B14, use the MAX function to find the highest value among the three monthly totals (you'll need to create formulas in row 10 for B10, C10, and D10 first)
  30. In cell B15, use the MIN function to find the lowest monthly total
  31. In cell B16, use the AVERAGE function to calculate the average monthly total
  32. Format cells B14:B16 with Accounting Number Format, 0 decimal places
  33. Note: You will need to create SUM formulas in cells B10, C10, and D10 to calculate monthly totals before you can find the MAX, MIN, and AVERAGE of these values.
  34. PART 7: Conditional Formatting (10 points)
  35. Apply conditional formatting to the Quarter Total column (E5:E9) to highlight values greater than $20,000 with a Light Red Fill and Dark Red Text
  36. Apply a different conditional format to the same range to highlight values less than $15,000 with a Yellow Fill
  37. PART 8: Performance Metrics Formatting (5 points)
  38. Format cell A19 with Font: Arial, Size: 12pt, Bold
  39. Format cell B20 with Accounting Number Format, 0 decimal places
  40. Format cell B21 as Percentage with 0 decimal places (it should display as 5%)
  41. Apply a Gold, Accent 4, Lighter 60% Fill to cells B20:B21
  42. PART 9: Chart Creation (15 points)
  43. Create a Column Chart showing the Quarter Total for each product category
  44. Use the data range A4:A9 and E4:E9 (Product Categories and Quarter Totals)
  45. Move the chart to a new sheet named "Sales Chart"
  46. Add the chart title: "Q4 2024 Sales by Category"
  47. Apply Chart Style 3 (or any professional style of your choice)
  48. Ensure the chart is easy to read with appropriate sizing
  49. PART 10: Sheet Management and Printing (5 points)
  50. Rename the main data sheet tab to "Q4 Analysis"
  51. Change the "Q4 Analysis" sheet tab color to Blue
  52. Set up the "Q4 Analysis" sheet to print in Landscape orientation
  53. Ensure all data fits on one page when printed (use Print Preview to verify)
  54. GRADING RUBRIC
