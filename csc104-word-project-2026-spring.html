MS Word Assignment: Community Newsletter

Follow the instructions in each part to build a formatted multi-page document in Microsoft Word.

Your Scenario

You are a volunteer coordinator for Lakewood Community Center. You have been asked to create a professional multi-page newsletter announcing the Center's upcoming annual events, programs, and season budget. The newsletter will be distributed digitally to community members and posted on the Center's bulletin board.

Use only the text provided in each part's Content block below. Do not add or generate your own content.

🍎 Mac Users: Where Word for Mac uses different names or menu locations, you will see a blue note like this one. Follow the Mac-specific instruction instead of the Windows one.

Open a new blank Word document and apply the following settings before entering any content.

  • Margins: Set all page margins to Narrow (0.5" on all sides) via Layout → Margins → Narrow.
  • Theme: Change the document theme to Facet via Design → Themes → Facet.
    Mac: Go to Design → Themes. If “Facet” does not appear, choose Mesh as an equivalent. Theme names can vary between Mac and PC versions.
  • Theme Colors: Change only the color scheme to Blue Warm via Design → Colors → Blue Warm.
    Mac: Go to Design → Colors. If “Blue Warm” is not listed, choose Blue II as the equivalent.
  • Page Border: Apply a Box page border at 3/4 pt line weight in the default (Automatic) color via Design → Page Borders.
  • Document Properties: Open File → Info → Properties and set the Subject field to: Community Newsletter

Add the framing elements for your document before entering any body content.

  • Cover Page: Insert the built-in “Whisp” cover page (Insert → Cover Page → Whisp) and fill in all four content controls:
    • Document Title: Lakewood Community Center
    • Document Subtitle: Annual Programs & Events Newsletter
    • Author: Your Name
    • Date: Select today's date using the date picker
    Mac: If “Whisp” is not available, use Ion or Banded as an alternative and fill in the same four controls.
  • Header — Different First Page: Turn on “Different First Page.” Starting on page 2, insert the text Lakewood Community Center — 2025 Newsletter, right-aligned.
    Mac: Double-click the top margin of page 2 to open the header area. In the Header & Footer tab, check Different First Page. Type the text and press ⌘R to right-align.
  • Footer — Centered Page Number: Insert a centered page number on pages 2+ via Insert → Page Number → Bottom of Page → Plain Number 2. Confirm the cover page has no page number.
    Mac: Insert → Header & Footer → Page Number → Bottom of Page → Plain Number 2. Verify “Different First Page” is still checked.

On the page after the cover page, type the content below, then apply each formatting task. Each task targets a distinct piece of text and is graded independently.

  • Title Line: Type Welcome to Our 2025 Programs! on its own line. Select it and set the font to Calibri Light, size 24 pt, centered. Apply All Caps via Home → Change Case → UPPERCASE (or the Font dialog → All caps checkbox, Ctrl+D / ⌘D).
  • Paragraph Shading: Select the entire first body paragraph beginning “We are thrilled to announce…” and apply a shading color of Blue, Accent 1, Lighter 60% via Home → Paragraph → Shading dropdown.
    Mac: Select the paragraph, click the Shading button (paint-bucket icon) on the Home tab. Choose the swatch at column 5, row 4 of the theme color palette (the light-blue Accent 1 variant). The tooltip name may differ but the grid position is the same.
  • Bulleted List: Select the four program name lines beginning “Youth Soccer League…” and apply a bulleted list via Home → Bullets.
  • Numbered List: Select the three registration step lines beginning “Visit our website…” and apply a numbered list via Home → Numbering. (The bulleted and numbered lists must use different styles and are graded independently.)
  • Underline: In the closing paragraph, select the exact phrase register by August 15 and apply underline formatting (Ctrl+U / ⌘U).
  • Italic: Select the phrase all ages and skill levels welcome in the programs paragraph and apply italic formatting (Ctrl+I / ⌘I).
  • Footnote: Click at the end of the sentence ending “…subject to change without notice.” and insert a footnote via References → Insert Footnote. Enter the following as the footnote text:
    Registration fees are non-refundable. Contact the front desk for exceptions.

Content — type this text into your Word document

Welcome to Our 2025 Programs!

We are thrilled to announce another exciting year of programs and events at Lakewood
Community Center. This newsletter outlines everything you need to know about our
upcoming offerings, registration procedures, and facility updates. All programs are
designed for all ages and skill levels welcome.

Programs offered this season include:
Youth Soccer League — Saturdays, starting September 6
Adult Fitness Boot Camp — Monday/Wednesday evenings
Senior Art Workshop — Tuesday afternoons, all levels
Family Movie Nights — Last Friday of each month, free admission

To register for any program, please follow these steps:
Visit our website at www.lakewoodcc.org/register
Complete the online form and submit payment
You will receive a confirmation email within 2 business days

Space is limited in all programs, so we encourage all interested community members to
register by August 15. Program schedules are subject to change without notice.

Add a visual element and a section break. Each task is independently graded.

  • Insert Picture: Below the bulleted list, add a blank centered line, then insert a picture of a community park or outdoor activity via Insert → Pictures → Online Pictures (search “community park”). Any appropriate image is accepted.
  • Resize Picture: Resize the picture to exactly 2.5 inches wide via Picture Format → Size → Width. Verify Lock aspect ratio is checked.
    Mac: Select the picture, go to Picture Format → Size or right-click → Size and Position. Enter 2.5" in the Width field with “Lock aspect ratio” checked.
  • Picture Style: Apply the “Soft Edge Rectangle” style from Picture Format → Picture Styles gallery. Click the More arrow (▿) to expand the gallery if needed.
    Mac: Style names are identical on Mac. Expand the gallery using the More arrow.
  • Text Wrapping: Set wrapping to Square via Picture Format → Wrap Text → Square. Drag the picture to the right side of the programs paragraph so text wraps to its left.
  • Page Break: Click after the closing paragraph and insert a manual page break with Ctrl+Enter / ⌘Enter to push the budget section to a new page.

On the new page, add the season budget section. Each task is independently graded.

  • Section Heading: Type 2025 Season Budget Overview on the first line. Apply the built-in Heading 1 style via Home → Styles → Heading 1. Do not manually change font or size — use the style only.
  • Insert Table: Below the heading insert a 5-row × 3-column table via Insert → Table and enter the data from the Content block below.
  • Table Style: Apply “Grid Table 5 Dark – Accent 1” from the Table Design tab → Table Styles gallery (click inside the table first to reveal this tab).
    Mac: Click inside the table to reveal the Table Design tab. If the exact style name is not available, choose the closest dark-accent equivalent and add a Word comment noting your choice.
  • Merge & Bold Total Row: Select all three cells in the bottom row and merge them via Table Layout → Merge Cells. Type Total Budget: $11,500, apply Bold, and center-align (Ctrl+E / ⌘E).
    Mac: Select the bottom-row cells → Table Layout → Merge Cells. Type, bold, and center with ⌘E.
  • Cell Alignment: Select data rows 2–4 (not the header or merged row) and center content both horizontally and vertically via Table Layout → Alignment → Align Center.
    Mac: Select rows 2–4, then click the center button (middle icon in the 3×3 alignment grid) in the Table Layout tab.

Content — data for budget table

Row 1 (Header):    Category    | Item Description                | Allocated Budget
Row 2:             Facilities  | Building maintenance & repairs  | $4,200
Row 3:             Programs    | Instructor fees & supplies      | $5,800
Row 4:             Events      | Marketing & community events    | $1,500
Row 5 (merge all): Total Budget: $11,500

Below the budget table, add a SmartArt process diagram, then apply a draft watermark to the entire document.

  • Insert SmartArt: Insert a “Basic Process” SmartArt graphic via Insert → SmartArt → Process → Basic Process.
    Mac: Same path: Insert → SmartArt → Process → Basic Process. The dialog is identical on Mac.
  • SmartArt Text: Enter one label per shape. Delete any extra shapes beyond three (select the shape and press Delete, or right-click → Delete Shape).
    • Shape 1: Plan Programs
    • Shape 2: Open Registration
    • Shape 3: Host Events
  • SmartArt Colors: With the SmartArt selected, go to SmartArt Design → Change Colors and choose “Colorful Range – Accent Colors 1 to 2”.
    Mac: Click the SmartArt, go to SmartArt Design → Change Colors. If the exact name differs, choose the first option under the “Colorful” heading.
  • Watermark: Apply the built-in “DRAFT 1” text watermark via Design → Watermark → DRAFT 1.
    Mac: Go to Design → Watermark. If only “Custom Watermark” is available, select it, choose Text watermark, type DRAFT, set orientation to Diagonal, and check Semi-transparent.

Each row is graded independently — a missed step in one row does not affect points in any other row.

Submission Instructions

After completing all parts in Microsoft Word, save your document as
Newsletter_YourLastName_YourFirstName.docx
for example: Newsletter_Smith_Jane.docx
Submit this single .docx file to the course assignment portal. Your instructor will open the Word file directly to verify all formatting.

Mac users: Use File → Save As and confirm the format is Word Document (.docx) — not .pages — before submitting.
