MS Word Assignment: Community Newsletter

Follow the instructions in each part to build a formatted multi-page document in Microsoft Word.

Your Scenario

You are a volunteer coordinator for Lakewood Community Center. You have been asked to create a professional multi-page newsletter announcing the Center's upcoming annual events, programs, and season budget. The newsletter will be distributed digitally to community members and posted on the Center's bulletin board.

Use only the text provided in each part's Content block below. Do not add or generate your own content.

 Mac Users: Some feature names differ between Word for Mac and Word for Windows. Where a name differs, a blue note like this one will show the Mac equivalent name to look for.

Open a new blank Word document and apply the following settings before entering any content.

  • Margins: Set all page margins to Narrow.
  • Theme: Change the document theme to Facet.
     Mac: If “Facet” is not available, use Mesh as the equivalent.
  • Theme Colors: Change the color scheme (not the full theme) to Blue Warm.
     Mac: If “Blue Warm” is not listed, use Blue II as the equivalent.
  • Page Border: Apply a Box page border at 3/4 pt line weight in the default (Automatic) color.
  • Document Properties: Set the Subject field to: Community Newsletter

Add the framing elements for your document before entering any body content.

  • Cover Page: Insert the built-in “Whisp” cover page and fill in all four content controls:
    • Document Title: Lakewood Community Center
    • Document Subtitle: Annual Programs & Events Newsletter
    • Author: Your Name
    • Date: Today’s date
     Mac: If “Whisp” is not available, use Ion or Banded as an alternative.
  • Header — Different First Page: The header must be absent on the cover page. Starting on page 2, the header must contain the text Lakewood Community Center — 2025 Newsletter, right-aligned.
  • Footer — Centered Page Number: Insert a centered page number in the footer on pages 2+. The cover page must have no page number.

On the page after the cover page, type the content below, then apply each formatting task. Each task targets a distinct piece of text and is graded independently.

  • Title Line: The line Welcome to Our 2025 Programs! must be formatted as Calibri Light, size 24 pt, centered, with All Caps applied.
  • Paragraph Shading: The entire first body paragraph beginning “We are thrilled to announce…” must have a background shading color of Blue, Accent 3, Lighter 60%.
     Mac: This color is the swatch at column 5, row 4 of the theme color palette. The tooltip name may differ slightly from Windows.
  • Bulleted List: The four program name lines beginning “Youth Soccer League…” must be formatted as a bulleted list.
  • Numbered List: The three registration step lines beginning “Visit our website…” must be formatted as a numbered list. The bulleted and numbered lists must use different list styles and are graded independently.
  • Underline: The exact phrase register by August 15 in the closing paragraph must be underlined.
  • Italic: The exact phrase all ages and skill levels welcome must be italicized.
  • Footnote: A footnote must be inserted at the end of the sentence ending “…subject to change without notice.” The footnote text must read:
    Registration fees are non-refundable. Contact the front desk for exceptions.

Content — type this text into your Word document

Welcome to Our 2025 Programs!

We are thrilled to announce another exciting year of programs and events at Lakewood
Community Center. This newsletter outlines everything you need to know about our
upcoming offerings, registration procedures, and facility updates. All programs are
designed for all ages and skill levels welcome.

Programs offered this season include:
Youth Soccer League — Saturdays, starting September 6
Adult Fitness Boot Camp — Monday/Wednesday evenings
Senior Art Workshop — Tuesday afternoons, all levels
Family Movie Nights — Last Friday of each month, free admission

To register for any program, please follow these steps:
Visit our website at www.lakewoodcc.org/register
Complete the online form and submit payment
You will receive a confirmation email within 2 business days

Space is limited in all programs, so we encourage all interested community members to
register by August 15. Program schedules are subject to change without notice.

Add a visual element and a section break. Each task is independently graded.

  • Insert Picture: Below the bulleted program list, insert an online picture of a community park or outdoor activity. Any appropriate, school-suitable image is accepted.
  • Resize Picture: The picture must be exactly 2.5 inches wide with aspect ratio locked (height adjusts proportionally).
  • Picture Style: Apply the “Soft Edge Rectangle” picture style.
  • Text Wrapping: Set the picture’s text wrapping to Square and position it on the right side of the programs paragraph so text wraps to its left.
  • Page Break: Insert a manual page break after the closing paragraph ( ... change without notice1.) to push the budget section to a new page.

On the new page, add the season budget section. Each task is independently graded.

  • Section Heading: The text 2025 Season Budget Overview must have the built-in Heading 1 style applied. Do not manually change the font or size — use the style only.
  • Insert Table: Below the heading, insert a 5-row × 3-column table and enter the data from the Content block below.
  • Table Style: Apply the “Grid Table 5 Dark – Accent 1” table style.
     Mac: If this exact style name is not available, choose the closest dark-accent equivalent and add a Word comment to your document noting the style you used.
  • Merge & Bold Total Row: All three cells in the bottom row must be merged into one cell containing the text Total Budget: $11,500 formatted as Bold and center-aligned.
  • Cell Alignment: The content in data rows 2–4 (not the header or merged row) must be centered both horizontally and vertically.

Content — data for budget table

Row 1 (Header):    Category    | Item Description                | Allocated Budget
Row 2:             Facilities  | Building maintenance & repairs  | $4,200
Row 3:             Programs    | Instructor fees & supplies      | $5,800
Row 4:             Events      | Marketing & community events    | $1,500
Row 5 (merge all): Total Budget: $11,500

Below the budget table, add a SmartArt process diagram, then apply a draft watermark to the entire document.

  • Insert SmartArt: Insert a “Basic Process” SmartArt graphic from the Process category.
  • SmartArt Text: The graphic must contain exactly three shapes with the following labels (delete any extra default shapes):
    • Shape 1: Plan Programs
    • Shape 2: Open Registration
    • Shape 3: Host Events
  • SmartArt Colors: The color scheme must be set to “Colorful Range – Accent Colors”.
     Mac: If the exact name differs, choose the first option under the “Colorful” heading in the Change Colors gallery.
  • Watermark: Apply the built-in “DRAFT 1” text watermark to the document.
     Mac: If only “Custom Watermark” is available, create a text watermark reading DRAFT with diagonal orientation and transparency at 15%.

Each row is graded independently — a missed step in one row does not affect points in any other row.
