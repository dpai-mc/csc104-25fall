CSC104: Computers and Computer Applications

2026 Spring: Hybrid Course - Tuesdays, 1:30 PM to 3:15 PM

Course Description

This course will present the computer as a problem-solving tool, with a focus on word processing, spreadsheet, presentation, email and group scheduling applications. Additional topics include an introduction to fundamental computer concepts such as operating systems, hardware, networking and graphics.

Student Learning Objectives

Upon successful completion of this course, students will be able to:

  • Use a word-processing application to create, format edit and save a multi-page document that includes features such as tables, ClipArt, borders, shading, headers and footers, as measured by exam questions, exercises and/or projects.
  • Use a presentation application to create, format, edit and save a presentation that includes features such as multiple slide layouts, consistent slide design, ClipArt, slide master, headers, footers, transitions and animations, as measured by exam questions, exercises and/or projects.
  • Use a spreadsheet application to create, format, edit and save a workbook that includes features such as formulas, functions, ClipArt, borders, shading, charts, headers and footers, as measured by exam questions, exercises, and/or projects.
  • Use an email and scheduling application to compose, reply and manage email as well as to create and schedule meetings using an integrated scheduler.

Success Tips for Online Learning

Time Management

  • Set aside dedicated study time each week (recommended: 6-9 hours per week)
  • Use a calendar to track assignment due dates
  • Start assignments early to avoid last-minute technical issues
  • Break larger projects into smaller, manageable tasks

Technical Preparation

  • Test your computer and internet connection before starting assignments
  • Keep software updated (Canvas, Microsoft Office, web browser)
  • Save work frequently and maintain backup copies
  • Have a backup plan for internet connectivity issues

Getting Help

  • Don't wait until the last minute to ask for help
  • Use discussion forums to connect with classmates
  • Contact the instructor with specific questions
  • Take advantage of MindTap tutorials and practice exercises
